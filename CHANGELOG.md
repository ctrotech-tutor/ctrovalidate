# Changelog

## [1.0.0] — 2026-06-08

### Added

- Initial release of the ctrovalidate docs site
- Documentation for all 6 standalone packages:
  - **ctrovalidate-core** — Platform-agnostic validation engine, 22 rules, 4 validation functions
  - **ctrovalidate-browser** — DOM adapter with declarative HTML-first API
  - **ctrovalidate-react** — Headless hook for React 18+
  - **ctrovalidate-vue** — Reactive composable for Vue 3
  - **ctrovalidate-svelte** — Store-based integration for Svelte
  - **ctrovalidate-next** — Server Action utilities for Next.js
- API reference, guide, advanced, and platform integration pages
- Hybrid schema system documentation (string, array, alias formats)
- Async validation with AbortController integration guide
- i18n and localization documentation
- Custom rules and testing guides
- Conditional validation with `data-ctrovalidate-if`
